# المساهمة في المشروع

شكراً لاهتمامك بالمساهمة في تطبيق الدردشة! نرحب بجميع المساهمات.

## خطوات المساهمة

### 1. Fork المشروع
انقر على زر Fork في أعلى الصفحة.

### 2. Clone الـ Repository
```bash
git clone https://github.com/YOUR_USERNAME/gkfcm.git
cd gkfcm
```

### 3. إنشاء فرع جديد
```bash
git checkout -b feature/your-feature-name
```

### 4. اجعل التغييرات
- اتبع معايير الكود
- أضف تعليقات واضحة
- اختبر تغييراتك

### 5. Commit التغييرات
```bash
git commit -m "Add: وصف التغيير بوضوح"
```

### 6. Push إلى الفرع
```bash
git push origin feature/your-feature-name
```

### 7. إنشاء Pull Request
افتح Pull Request مع وصف واضح للتغييرات.

## معايير الكود
- استخدم TypeScript
- اتبع نمط React Hooks
- اكتب tests للميزات الجديدة
- استخدم Prettier للـ formatting

## الإبلاغ عن الأخطاء
إذا وجدت خطأ، يرجى فتح Issue مع:
- وصف واضح للمشكلة
- خطوات إعادة إنتاج الخطأ
- سلوك متوقع مقابل السلوك الفعلي
