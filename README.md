# تطبيق الدردشة الكتابية

## الوصف
تطبيق دردشة كتابية جميل ومتحرف يوفر تجربة محادثة فورية وآمنة.

## الميزات
- 💬 دردشة فورية في الوقت الحقيقي
- 👥 دعم المحادثات الفردية والجماعية
- 🔒 تشفير آمن للرسائل
- 📱 واجهة مستخدم سهلة وجميلة
- 🌍 دعم اللغات المتعددة

## البدء السريع

### المتطلبات
- Node.js 16+
- npm أو yarn

### التثبيت
```bash
git clone https://github.com/ynjar2336-creator/gkfcm.git
cd gkfcm
npm install
```

### التطوير
```bash
npm run dev
```

### البناء
```bash
npm run build
```
